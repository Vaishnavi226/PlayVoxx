## PlayVoxx

PlayVoxx is a simple, stylish web-based music player interface. It features trending songs, popular artists, and a modern, responsive design inspired by popular music streaming platforms.

## Features

Trending songs section with album art and play buttons
Popular artists showcase
Search bar for finding songs
Playlist creation prompt
Responsive layout with sidebar and footer
Font Awesome icons and Google Fonts integration

## Project Structure

index.html — Main landing page with trending songs and artists
player.html — (Presumed) Player interface (not provided)
script.js — JavaScript for interactivity (e.g., play buttons)
style.css — Custom styles for layout and design
Images and audio files (e.g., .jpeg, .mp3) — Used for song cards and artist cards

## Getting Started

Clone or download the repository.
Ensure all referenced images and audio files (e.g., Bairan.jpeg, Bairan.mp3) are present in the project directory.
Open index.html in your web browser.

## Dependencies

Font Awesome 7.0.1 for icons
Google Fonts: Inter

## Customization

Add or update songs by editing the song cards in index.html.
Add new artists in the "Popular Artists" section.
Update styles in style.css as needed.
